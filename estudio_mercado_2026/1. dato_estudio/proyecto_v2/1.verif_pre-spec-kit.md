# ✅ Fase 0: Verificación Pre-Spec-Kit (Opción A Confirmada)

¡Perfecto! Empezamos con lo más importante: **verificar que todo está en su sitio antes de mover una sola línea de código con Spec-Kit**. Esto evita errores en cadena.

Ejecuta estos comandos en **PowerShell** dentro de la carpeta de tu proyecto (`ia-mercado-laboral/`).

---

## 📋 BLOQUE 1: Verificar Herramientas Instaladas

```powershell
# 1. Verificar Spec-Kit
specify --help

# 2. Verificar Bun
bun --version

# 3. Verificar Git
git --version
```

**✅ Éxito esperado:**
*   `specify`: Versión 1.x.x
*   `bun`: Versión 1.x.x
*   `git`: Versión 2.x.x

---

## 📂 BLOQUE 2: Verificar Archivos Críticos

```powershell
# 1. Verificar configuración Spec-Kit
Test-Path .specify/constitution.md && Write-Host "✓ constitution.md existe" -ForegroundColor Green || Write-Host "✗ FALTA constitution.md" -ForegroundColor Red
Test-Path .specify/specifications.md && Write-Host "✓ specifications.md existe" -ForegroundColor Green || Write-Host "✗ FALTA specifications.md" -ForegroundColor Red

# 2. Verificar Diseño
Test-Path DESIGN.md && Write-Host "✓ DESIGN.md existe" -ForegroundColor Green || Write-Host "✗ FALTA DESIGN.md" -ForegroundColor Red

# 3. Verificar Skills
Test-Path .agents/skills && Write-Host "✓ Carpeta skills existe" -ForegroundColor Green || Write-Host "✗ FALTA carpeta skills" -ForegroundColor Red

# 4. Verificar 404 (Crítico para GitHub Pages)
Test-Path public/404.html && Write-Host "✓ public/404.html existe" -ForegroundColor Green || Write-Host "✗ FALTA public/404.html" -ForegroundColor Red
Test-Path src/pages/404.astro && Write-Host "✓ src/pages/404.astro existe" -ForegroundColor Green || Write-Host "✗ FALTA src/pages/404.astro" -ForegroundColor Red

# 5. Verificar Astro Config
Test-Path astro.config.ts && Write-Host "✓ astro.config.ts existe" -ForegroundColor Green || Write-Host "✗ FALTA astro.config.ts" -ForegroundColor Red
```

---

## 🏗️ BLOQUE 3: Verificar Build Local (CRÍTICO)

```powershell
# 1. Instalar dependencias (por si acaso)
bun install

# 2. Ejecutar build
bun run build

# 3. Verificar que se generó dist/
Test-Path dist/ && Write-Host "✓ Carpeta dist/ generada" -ForegroundColor Green || Write-Host "✗ FALTA carpeta dist/" -ForegroundColor Red

# 4. Verificar que la 404 está en el build
Test-Path dist/404.html && Write-Host "✓ dist/404.html existe" -ForegroundColor Green || Write-Host "✗ FALTA dist/404.html" -ForegroundColor Red

# 5. Verificar index.html
Test-Path dist/index.html && Write-Host "✓ dist/index.html existe" -ForegroundColor Green || Write-Host "✗ FALTA dist/index.html" -ForegroundColor Red
```
