## 📋 Paso 1: Verificar Herramientas (Copiar y Pegar)

Ejecuta este bloque de comandos. He añadido comentarios para que sepas qué hace cada uno.

```powershell
# 1. Verificar Spec-Kit (¿Está instalado?)
Write-Host "🔍 Verificando Spec-Kit..." -ForegroundColor Cyan
specify --version
if ($LASTEXITCODE -ne 0) { Write-Host "❌ ERROR: Spec-Kit no está instalado o no está en el PATH" -ForegroundColor Red }

# 2. Verificar Git (¿Está inicializado?)
Write-Host "`n🔍 Verificando Git..." -ForegroundColor Cyan
git status
if ($LASTEXITCODE -ne 0) { Write-Host "❌ ERROR: Git no está inicializado. Ejecuta 'git init'" -ForegroundColor Red }

# 3. Verificar Bun y Dependencias
Write-Host "`n🔍 Verificando Bun..." -ForegroundColor Cyan
bun --version
bun install --frozen-lockfile

# 4. Verificar Archivos Críticos
Write-Host "`n🔍 Verificando estructura de archivos..." -ForegroundColor Cyan
$files = @(
    ".specify/constitution.md",
    ".specify/specifications.md",
    "DESIGN.md",
    "public/404.html",
    "src/data/randstad-catalunya.json",
    "astro.config.ts"
)

foreach ($file in $files) {
    if (Test-Path $file) {
        Write-Host "  ✅ $file" -ForegroundColor Green
    } else {
        Write-Host "  ❌ FALTA: $file" -ForegroundColor Red
    }
}

# 5. Prueba de Build Local (¿Compila sin errores?)
Write-Host "`n🔍 Ejecutando build de prueba..." -ForegroundColor Cyan
bun run build

# 6. Verificar Output del Build
if (Test-Path "dist/index.html") {
    Write-Host "  ✅ Build exitoso: dist/index.html generado" -ForegroundColor Green
} else {
    Write-Host "  ❌ ERROR: Build fallido o dist/ vacío" -ForegroundColor Red
}

if (Test-Path "dist/404.html") {
    Write-Host "  ✅ 404 correcta: dist/404.html generado" -ForegroundColor Green
} else {
    Write-Host "  ❌ ERROR: dist/404.html no existe (Crítico para GitHub Pages)" -ForegroundColor Red
}
```
