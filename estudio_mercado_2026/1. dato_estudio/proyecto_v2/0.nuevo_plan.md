#### **Nueva Especificación 6: Página de Fuentes**

```markdown
🎯 Especificación 6: Página de Fuentes y Referencias

Qué
Una página dedicada que muestra todas las fuentes utilizadas en el estudio.

Por qué
- Credibilidad académica (target sin estudios universitarios necesita confiar)
- Transparencia (requisito original del proyecto)
- Permitir verificación de datos
- Cumplir con fuentes oficiales (gubernamentales, universidades, OCDE)

Requisitos
- Tabla responsive (desktop: tabla, mobile: cards)
- Columnas: Nombre estudio (negrita), Tipo (económico/empleo), Fuente (organismo), Descripción (max 40 chars), Enlace (markdown)
- Filtros opcionales: por tipo de fuente (gubernamental, universidad, fundación)
- Diseño coherente con DESIGN.md (paleta, tipografía, asimetría)
- Accesibilidad WCAG AA

Datos
- Fuentes obligatorias: Randstad, ECB, BLS, Tomsk Polytechnic
- Fuentes sugeridas: INE, Idescat, OECD, ILO
- Formato: JSON en `src/data/fuentes.json`

Diseño
- Mismo sistema que tablas existentes (Table.astro)
- Iconos SVG por tipo de fuente
- Enlaces abren en nueva pestaña (target="_blank")
```

#### **Ejemplo de Comando Spec-Kit para Fuentes**

```
/speckit.specify
Crear página de fuentes y referencias del estudio.

Requisitos:
- Página accesible desde navegación principal (enlace "Fuentes")
- Tabla responsive: desktop (tabla clásica), mobile (cards apiladas)
- Columnas: 
  * Nombre del estudio (negrita, H3)
  * Tipo (económico/empleo/metodología)
  * Fuente (organismo: INE, OCDE, Universidad...)
  * Descripción (máximo 40 caracteres)
  * Enlace (formato markdown, abre en nueva pestaña)
- Filtros opcionales por categoría (gubernamental, universidad, fundación)
- Datos cargados desde src/data/fuentes.json
- Diseño coherente con DESIGN.md (paleta terracota/verde/crema)
- Accesibilidad WCAG AA (contraste 4.5:1, focus visible)
- Iconos SVG por tipo de fuente (🏛️ gubernamental, 🎓 universidad, 🏢 fundación)

Skills a utilizar:
- astro (estructura de páginas)
- tailwind-css-patterns (referencias/responsive-design.md)
- frontend-design (UI no-IA, asimetría)
- accessibility-wcag (WCAG AA)
```

